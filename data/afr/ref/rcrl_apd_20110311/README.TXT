Notes:

1. This dictionary was developed using the Lwazi Afrikaans dictionary as base. 
   For more info see http://www.meraka.org.za/lwazi/pd.php

2. The following diphthongs are modelled as phoneme combinations:
   'ai'		/ a i /
   'aai'	/ A: i /
   'ooi'	/ u@ i /
   'oi'		/ O i /
   'oei'	/ u i /
   'eeu'	/ i u /
   Depending on the application, these combinations should be re-mapped to diphthongs.

3. Vowel transitions are not modelled explicitly
   For example:
   viool	is modelled as 	f i o: l		and not as	f i j o: l
   italiaans	is modelled as 	i t a l i A: n s	and not as	i t a l i h A: n s

4. Hyphens are treated as word boundaries
